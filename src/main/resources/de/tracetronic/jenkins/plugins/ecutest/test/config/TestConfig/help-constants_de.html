Definition von globalen Konstanten bestehend aus einem eindeutigen Namen und einem Wert. Numerische Wertangaben werden automatisch in ein Python Integer-Literal umgewandelt, sonst in ein Python String-Literal.
Falls die globale Konstante nicht existiert, wird sie erstellt und in der aktuellen Testkonfiguration gespeichert.
Achtung:Ab ECU-TEST Version 2022.3 werden keine ungültigen Steuerzeichen in Strings mehr anerkannt! D.h. insbesondere Backslashes in Windows-Pfaden müssen hier maskiert werden (C:\\mein\\pfad statt C:\mein \pfad, damit 'Backslash - nächstes Zeichen') nicht als Escape-Sequenz interpretiert wird und es in ECU-TEST zu keinem Fehler kommt.
