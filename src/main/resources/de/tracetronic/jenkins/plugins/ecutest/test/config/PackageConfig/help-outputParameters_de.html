Definition von ECU-TEST Package-Rückgabe-Parametern bestehend aus einem eindeutigen Namen.
Definierte Package-Rückgabe-Parameter werden in die zugehörigen Umgebungsvariablen geschrieben, nachdem die Testausführung beendet ist.
Die Umgebungsvariable entspricht folgender Syntax: TT_TEST_RETVAL_<VARIABLE NAME>_<INDEX>
